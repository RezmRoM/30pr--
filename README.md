# Новая жизнь - Сервис поиска потерянных животных

Веб-платформа для помощи в поиске потерянных домашних животных в Ярославле. Проект разработан с использованием чистого HTML, CSS и JavaScript без использования фреймворков.

## Структура проекта

```
project/
│
├── css/
│   ├── style.css
│   └── media.css
│
├── js/
│   └── main.js
│
├── img/
│   └── logo.svg
│
├── index.html
├── login.html
├── register.html
├── profile.html
├── add-pet.html
├── search.html
├── orders-management.html
└── pet.html
```

## Запуск проекта

Вы можете запустить проект одним из двух способов:

### 1. Используя Python

#### Требования
- Python 3.x

#### Шаги для запуска

1. Откройте терминал и перейдите в директорию проекта:
```bash
cd путь/к/проекту
```

2. Для Python 3:
```bash
python -m http.server 7000
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:7000
```

### 2. Используя Node.js

#### Требования
- Node.js (версия 12.x или выше)

#### Шаги для запуска

1. Откройте терминал и перейдите в директорию проекта:
```bash
cd путь/к/проекту
```

2. Если вы запускаете проект впервые, установите зависимости:
```bash
npm install
```

3. Запустите сервер:
```bash
node server.js
```

4. Откройте браузер и перейдите по адресу:
```
http://localhost:7001
```
